# Okta oAuth Go Authorization App

## Setup

In order for the Okta Authentication to work, you'll need developer credentials which the this application gathers from a file in the root directory called `secrets.json`

## Requirements

- Go 1.11
- Node.js v8.7.0 +

## Run Server

- **go get -v ./...**
- **go build**
- **./server**

## Run Web

- **npm i**
- **npm start**
